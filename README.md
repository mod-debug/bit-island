## What is Bit Island?

Bit Island is a DeFi toolkit built natively on Bitcoin Layer 1 using [OPNet](https://opnet.org) smart contracts. No bridges. No sidechains. Real Bitcoin.

---

## Modules

### Revenue Vault
Auto-compound vault for OP20 tokens. Deposit, earn, withdraw. On-chain yield directly on Bitcoin.

### OTC Trading
Peer-to-peer token trading without a centralized order book. Create offers, accept deals, fully trustless.

### Vesting Dashboard
On-chain token vesting with cliff and linear release schedules. Full transparency for teams and investors.

### NFT Ecosystem
OP721 NFT minting, collection management, and marketplace offers — all on Bitcoin L1.

### Launchpad
Deploy OP20 tokens and launch projects directly from the interface.

---

## Tech Stack

- **Smart contracts**: OPNet (OP_NET) — Bitcoin L1 Tapscript
- **Frontend**: React + TypeScript + Vite
- **Wallet**: OPNet WalletConnect
- **Network**: Bitcoin Mainnet / Testnet via OPNet
